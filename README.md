FitPlanner

Android-приложение для планирования и контроля тренировок между тренером и спортсменом.

 Технологии
- Язык: Kotlin
- Архитектура: Fragment-based UI
- БД: SQLite (Room ORM)
- Асинхронность: Kotlin Coroutines
- UI: Material Design

 Роли
- Атлет: просмотр тренировок, фильтрация, отметка выполнения
- Тренер: создание планов, просмотр атлетов, фильтрация
- Админ: назначение тренера спортсмену

 Установка
1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Запустите на эмуляторе или устройстве

 Развитие
- Backend (REST API)
- Push-уведомления
- История прогресса
